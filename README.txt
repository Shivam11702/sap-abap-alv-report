Project Title:
Custom ALV Report using SAP ABAP

Project Description:
This project demonstrates development of a Custom ALV (ABAP List Viewer) Report using SAP ABAP programming language.

The report retrieves structured data from SAP database table VBAK and displays output in ALV Grid format.

The ALV report provides interactive features such as:

- column sorting
- data filtering
- structured tabular display
- improved data readability

Tools and Technologies Used:
SAP S/4HANA
ABAP Programming Language
SE38 ABAP Editor
ALV Grid Display
SAP Data Dictionary (SE11)

Tables Used:
VBAK – Sales document header data

Project Objective:
To demonstrate practical implementation of ABAP concepts such as:

- SELECT query
- internal tables
- field catalog
- ALV report output
